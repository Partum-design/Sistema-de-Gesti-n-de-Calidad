# INDUSECC Superadmin

Carpeta principal ordenada para trabajar mejor el proyecto.

## Estructura recomendada

```text
indusecc-react-superadmin/
|- backend/                 <- codigo principal del backend
|- frontend/                <- codigo principal del frontend
|- _local/                  <- archivos locales, pesados o sensibles
|- Indusecc-Backend/        <- respaldo temporal bloqueado por Windows
|- indusecc-react-Frontend/ <- respaldo temporal bloqueado por Windows
|- .gitignore
`- README.md
```

## Que usar normalmente

- Trabaja en `backend/`
- Trabaja en `frontend/`

## Que se dejo en `_local/`

- `.env`
- `node_modules`
- `node_modules_old`
- `uploads`
- builds temporales

## Nota

Las carpetas `Indusecc-Backend/` e `indusecc-react-Frontend/` quedaron como respaldo temporal porque algunos archivos estaban bloqueados por Windows o OneDrive al momento de moverlos.
